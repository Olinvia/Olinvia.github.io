小林翻墙记

2020-11-24 小林日记

翻过那堵墙，听见世界的故事！

​ 翻墙是怎么回事呢？翻墙相信大家都很熟悉，但是翻墙是怎么回事呢，下面就让小编带大家一起了解吧。翻墙，其实就是翻过那堵墙，大家可能会很惊讶墙怎么翻呢？但事实就是这样，小编也感到非常惊讶。这就是关于翻墙的事情了，大家有什么想法呢，欢迎在评论区告诉小编一起讨论哦！

​ 在此，我们不讨论任何跟翻墙知识有关的东西，因为我有 < b> 钞能力 。

​ 1. 首先，打开 SudaCloud，注册网址为：https://v2.suda.moe/auth/register，它长这样：

​ 2. 进行注册后登录，先进入我的钱包，使用钞能力为钱包充值。

​ 3. 然后进入商店，用钱包里的钱进行购买。

​ 4. 订阅完成后，下载 Clash for Windows ，下载地址为：https://cowtransfer.com/s/221489174e8642

​ 5. 在网站主页上的快速使用中点击 clash 订阅，点击复制 clash 订阅。

​ 6. 打开 Clash for Windows，跳转到 Profiles 页面，粘贴 clash 订阅，点击 download。

​ 7. 最后 General->System Proxy，打开 System Proxy 即可。

​ 翻过去咯！

下午 4 点，我望着一碧如洗的天空，喃喃自语：

“国外的月亮真圆啊。”
