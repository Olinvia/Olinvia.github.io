数据类型和变量

操作系统将时间分成很多细小的时间片，一个时间片给一个程序用，并且频繁地在程序间切换，给程序造成整个机器资源都归它使用的错觉。

声明一个变量，如 int a，其实就是在内存中分配了一块空间，这块空间存放 int 数据类型，a 指向这块内存空间所在的位置，通过对 a 操作即可操作 a 指向的内存空间。
