类的扩展

接口

在某些情况下，类型并不重要，重要的是能力。

在 Java8 之前，接口方法都是 public abstract。

抽象类

这个概念介于接口和类之间。

只有子类才知道如何实现的方法，一般被定义为抽象方法。

内部类

  • 静态内部类（static 修饰）
  • 成员内部类（无 static 修饰）
  • 方法内部类
