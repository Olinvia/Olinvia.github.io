包装类与基础类的转换：

String 类

可以直接用 +。

看似在修改，其实创建了新对象。

如果频繁地修改 String 内容，应该考虑 StringBuilder 和 StringBuffer。

Array 类

可以使用 Arrays.toString (arr) 来输出 arr 的各个元素。

创建多维数组时，第一维的长度可以指定，其他维不需要指定。

sort 采用的是双枢轴快速排序（基本类型）和 TimSort（对象）。

Date 类
